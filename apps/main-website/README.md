# main-website

To get started:
1. run `pnpm install`
2. start payload, and make sure that you have an account.
3. enable the api key on that account and save the changes in payload
4. copy sample.env to .env and paste the api key in the .env
5. run `pnpm dev` to start the website
6. visit `localhost:4321` in the browser to see the webpage
